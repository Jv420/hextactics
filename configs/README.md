# HexTactics Config Pack

Deze map bevat start-configs voor **HexTactics** op **Paper 1.21.11**.

## Gebruik

Kopieer de bestanden naar je servermap/pluginmap en pas wachtwoorden, tokens en IP's aan.

## Belangrijk

- Zet nooit echte wachtwoorden of tokens in GitHub.
- Gebruik deze configs als basis, niet als definitieve productie-secrets.
- Start na elke plugin-groep opnieuw en check console errors.

## Mappen

```txt
configs/
├─ server/
├─ luckperms/
├─ essentials/
├─ minimotd/
├─ geyser/
├─ deluxemenus/
├─ tab/
├─ hextacticsstore/
├─ chunky/
├─ worldguard/
└─ discordsrv/
```
